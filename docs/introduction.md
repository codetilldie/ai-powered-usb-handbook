# 项目简介 (Introduction)

## 项目愿景

USB (Universal Serial Bus) 接口已经渗透到我们数字生活的方方面面，从手机充电到高速数据传输，再到外接显卡扩展。然而，随着技术的迭代，USB 的命名规范变得异常混乱（例如 USB 3.2 Gen 2x2），接口形态与协议功能也经常让人混淆（Type-C 接口一定支持快充吗？）。

**AI-Powered USB Handbook** 致力于成为一份**清晰、准确、易懂**的 USB 技术百科全书。我们利用 AI 辅助整理繁杂的技术标准，并经过人工校验，为不同背景的读者提供有价值的信息。

## 适用人群

- **普通消费者**：想要搞清楚买什么线才能给手机满速充电，或者为什么新买的硬盘插在电脑上速度很慢。
- **数码爱好者**：对雷电 4、USB4 等前沿技术感兴趣，想要了解技术细节和硬件选购指南。
- **开发者与工程师**：需要快速查阅 USB 协议规范、引脚定义或供电标准的技术参考。

## 手册结构

本手册主要包含以下几个核心板块：

1.  **接口形态**：介绍物理接口的外观与特性。
2.  **传输协议**：详解从 USB 1.0 到 USB4 的速率与标准演进。
3.  **雷电技术**：深入解析 Intel 主导的 Thunderbolt 技术及其与 USB 的关系。
4.  **充电与供电**：揭秘 USB PD 快充、PPS 以及线材对充电功率的影响。
5.  **常见问题**：解答日常使用中的高频疑惑。

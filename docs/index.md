# AI-Powered USB Handbook

欢迎来到 **AI-Powered USB Handbook**！

AI-Powered USB Handbook 是一个利用人工智能生成深入浅出的 USB 接口与协议解读的开源项目。无论你是对 USB 3.2 Gen 2x2 这种复杂的命名规则感到困惑，还是分不清 Thunderbolt 4 (雷电 4) 与 全功能 Type-C 的区别，本手册都能为你清晰解析那些决定传输速率与充电效率的关键标准。
